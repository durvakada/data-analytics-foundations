# SQL Notes 

## SELECT 
- What: Choose columns from a table 
- Why : To see required data 
- Example :
SELECT name, score FROM students;

## WHERE 
- What: Filter rows
- Why : Focus on relevant records
- Example: 
SELECT * FROM students WHERE score > 70;

## ORDER BY 
- What: Sort results
- Why : Ranking and ordering
- Example:
SELECT * FROM students ODER BY score DESC;
